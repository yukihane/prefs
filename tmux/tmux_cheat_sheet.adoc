== 枠の用語

セッション > ウィンドウ > ペイン

== リスト
|===
|セッション| `tmux ls`
|ウィンドウ| `^b s`
|ペイン| `^b q`
|===

== 新規作成
|===
|セッション| `^b :new`
|ウィンドウ| `^b c`
|ペイン/右に作成| `^b %`
|ペイン/下に作成| `^b "`
|===

== 終了
|===
|セッション終了| `tmux kill-session`
|セッション終了 セッション指定| `tmux kill-session -t (セッション名)`
|ウィンドウ終了|`^b &`
|===

== 移動
|===
|ウィンドウ|前へ| `^b p`
|ウィンドウ|次へ| `^b n`
|ウィンドウ|番号指定| `^b ,`
|===

== よく使うもの
|===
|ウィンドウの名前変更| `^b ,`
|===
