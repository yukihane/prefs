# tmux チートシート

プレフィックスキー: **Ctrl+a** （デフォルトの Ctrl+b から変更済み）

## 用語

セッション > ウィンドウ > ペイン

## セッション

| 操作 | キー / コマンド |
|---|---|
| 新規作成 | `tmux new -s セッション名` |
| 一覧表示 | `tmux ls` |
| アタッチ | `tmux a -t セッション名` |
| デタッチ | `C-a d` |
| セッション一覧 (tmux内) | `C-a s` |
| セッション終了 | `tmux kill-session -t セッション名` |

## ウィンドウ

| 操作 | キー |
|---|---|
| 新規作成 | `C-a c` |
| 次へ | `C-a n` |
| 前へ | `C-a p` |
| 番号指定で移動 | `C-a 番号` |
| 名前変更 | `C-a ,` |
| 左へ入れ替え | `C-a <` |
| 右へ入れ替え | `C-a >` |
| 終了 | `C-a &` |

## ペイン

| 操作 | キー |
|---|---|
| 左右に分割 | `C-a \|` |
| 上下に分割 | `C-a -` |
| 左に移動 | `C-a h` |
| 下に移動 | `C-a j` |
| 上に移動 | `C-a k` |
| 右に移動 | `C-a l` |
| ペイン番号を表示 | `C-a q` |
| ペインを閉じる | `C-a x` |
| 他ウィンドウのペインを取り込む | `C-a J` |
| ペインをウィンドウに分離 | `C-a !` |

## コピーモード (vi キーバインド)

| 操作 | キー |
|---|---|
| コピーモード開始 | `C-a [` |
| 選択開始 | `v` |
| コピー＆終了 (クリップボード) | `Enter` |

## その他

| 操作 | キー / コマンド |
|---|---|
| 設定リロード | `C-a r` |
| Ctrl+a をアプリに送る | `C-a C-a` |
| コマンドモード | `C-a :` |

## 主な設定値

| 設定 | 値 |
|---|---|
| マウス操作 | 有効 |
| ウィンドウ番号開始 | 1 |
| ペイン番号開始 | 1 |
| ステータスバー位置 | 上部 |
| Escキー遅延 | 0ms |
| スクロールバック行数 | 50000 |
| ウィンドウ番号の自動詰め | 有効 |
| ターミナル | tmux-256color (TrueColor対応) |
| allow-passthrough | 有効 |
| コピーモード キーバインド | vi |
